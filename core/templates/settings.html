Help
Finding Your Tautulli API Key
  1. Open Tautulli
  2. Go to Settings > Web Interface
  3. Enable API (if not already enabled)
  4. Copy the API Key
Finding Plex Library Section IDs
  1. In Tautulli, go to Libraries
  2. The section ID is shown in the URL when you click on a library
  3. Example: http://localhost:8181/library?section_id=1 (section ID is 1)
Date Format

All dates should be in YYYY-MM-DD format. You can use "current" for the end date to use the current date.
